# Neovim 

This is my Neovim configuration forked from [Kickstart.nvim](https://github.com/nvim-lua/kickstart.nvim)

# Keybinds

## Normal Mode

### General
- `<leader>w` -> Write File
- `<leader>e` -> Open File Explore -> Open File Explorer


### Lsp
- `<Enter>` -> Accept suggestion
- `<Tab>` -> Next suggestion
- `<Shift><Tab>` -> Previous suggestion
